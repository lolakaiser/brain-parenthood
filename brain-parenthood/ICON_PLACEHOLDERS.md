## How to Update Icons

When you want to change an icon, tell me:

1. **The placeholder name** (e.g., `[ICON-ASSESS]`)
2. **What icon/image you want to use** - describe it or provide a file

### Examples:
- "Change `[ICON-ASSESS]` to a clipboard icon"
- "Change `[LOGO-BP]` to use this image file: logo.png"
- "Change `[ICON-GAIN-1]` to a brain with waves icon"

I'll update the code accordingly.
